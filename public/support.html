Harmony AI — Support

This page provides installation, usage, and support information for Harmony AI.

Installation

Harmony AI is distributed as a Windows EXE installer. Installation requires user interaction and password entry. Silent installation is not supported.

System Requirements

  • Windows 10 / 11 (64-bit)
  • Minimum 8 GB RAM recommended

Installer Return Codes

  • 0 — Installation successful
  • 1602 — Installation cancelled by user
  • 1618 — Installation already in progress
  • 1638 — Application already installed
  • 3010 — Restart required
